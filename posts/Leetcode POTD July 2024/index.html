5 July

  • Find the Minimum and Maximum Number of Nodes Between Critical Points

  • Solution

    • Traverse the list and keep track of min and max distance using prev_ind, first_ind and curr_ind

  • Code

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public int[] nodesBetweenCriticalPoints(ListNode head) {
        int min = Integer.MAX_VALUE;
        int max = Integer.MIN_VALUE;
        int prev = head.val;
        int prev_ind=-1;
        int first_ind = -1;
        int curr_ind = 1;
        head = head.next;
        int cnt = 0;
        while(head.next!=null){
            if((head.val>prev && head.val>head.next.val) |(head.val<prev && head.val<head.next.val)){
                cnt++;
                if(prev_ind==-1){
                    prev_ind = curr_ind;
                    first_ind = curr_ind;
                }
                else{
                    min = Math.min(min, curr_ind-prev_ind);
                    max = Math.max(max, curr_ind-first_ind);
                    prev_ind = curr_ind;
                }
            }
            prev = head.val;
            head = head.next;
            curr_ind++;
        }
        if(cnt<2){
            return new int[]{-1,-1};
        }
        return new int[]{min,max};
    }
}

4 July

  • Merge Nodes in Between Zeros

  • Solution

    • Keep a variable sum
    • Add the elements until you find 0
    • Add new node with value as sum and set sum to 0

  • Code

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode mergeNodes(ListNode head) {
        ListNode new_head = new ListNode();
        ListNode tmp = new_head;
        int sum = 0;
        head = head.next;
        while(head!=null){
            sum+=head.val;
            if(head.val==0){
                tmp.next = new ListNode(sum);
                tmp = tmp.next;
                sum=0;
            }
            head = head.next;
        }
        return new_head.next;
    }
}

3 July

  • Minimum Difference Between Largest and Smallest Value in Three Moves

  • Solution

    • There are 4 cases:-
      • Change the value of 3 minimum values
      • 2 min and 1 max
      • 1 min and 2 max
      • 3 max
    • The result will be min difference of all the cases.

  • Code

class Solution {
    public int minDifference(int[] nums) {
        if(nums.length<=4){
            return 0;
        }
        Arrays.sort(nums);
        int ans = nums[nums.length-4]-nums[0];
        for(int i=1;i<=4;i++){
            ans = Math.min(ans, nums[nums.length-i]-nums[4-i]);
        }
        return ans;
    }
}

  • Intersection of Two Arrays II

  • Solution

    • 1st approach is to use two hashamps for each array
    • Find intersection of those two using keys(map.keySet()) and intersection of those two sets(set = set.retainAll(s2));
    • 2nd approch is to make an array of length 1001(max num can be 1000)
    • While traversing nums1 increment the counter in cnt array
    • While traversing nums2 if cnt of num is 0 then continue else add num to ans and decrease the cnt by 1

  • Code

class Solution {
    public int[] intersect(int[] nums1, int[] nums2) {
        int[] cnt = new int[1001];
        
        for(int i=0;i<nums1.length;i++){
            cnt[nums1[i]]++;
        }
        ArrayList<Integer> arr = new ArrayList<>();
        for(int i=0;i<nums2.length;i++){
            if(cnt[nums2[i]]!=0){
                arr.add(nums2[i]);
                cnt[nums2[i]]--;
            }
        }
        int[] ans = new int[arr.size()];
        for(int i=0;i<ans.length;i++){
            ans[i] = arr.get(i);
        }
        return ans;

    }
}

1 July

  • Three Consecutive Odds

  • Solution

    • Keep a count if the currrent num is odd increase it and check if it is equal to 3 then return true
    • Otherwise set it back to 0

  • Code

class Solution {
    public boolean threeConsecutiveOdds(int[] arr) {
        int cnt = 0;
        for(int i=0;i<arr.length;i++){
            if(arr[i]%2==1){
                cnt++;
            }
            else{
                cnt=0;
            }
            if(cnt==3){
                return true;
            }
        }
        return false;
    }
}
